# COMP2240A3
Java Virtual Memory Management Simulator

## Supported frameworks
* Java 8

## Setup

Clone, Download or Extract all files to a folder
### Note: this repo was setup to run in IntelliJ, if extracting from ZIP, contents will all be in a single directory. All terminal commands should be run from that directory.
```java
// Compile
javac *.java

```

## Usage

```java
// Run Program with Input files params
java c3063467A3 Process1.txt Process2.txt ... Process(n).txt
```
* Where Process.txt is your designated input filename - path names are relative so file must be in root directory.

## Output
Simulation data is displayed in terminal.  Data is also written to output.txt which will be generated at runtime.
Please ensure your working directory is not write protected.
